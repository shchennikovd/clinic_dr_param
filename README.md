# 📖 О проекте

Я принимаю участие в разработке коммерческого сайта частной медицинской клиники в роли Frontend Developer.

Основная задача — разработка новых страниц по дизайн-макетам, реализация адаптивной верстки и создание пользовательского интерфейса с последующей передачей кода для интеграции в существующий проект.

Каждая папка в репозитории представляет собой отдельную коммерческую задачу.

---

# 👨‍💻 Моя роль

В рамках проекта я отвечаю за frontend-разработку новых страниц и компонентов.

Работа ведется по готовым дизайн-макетам с соблюдением существующего визуального стиля проекта.

---

# 🚀 Что входит в мои задачи

* разработка новых страниц по дизайн-макетам;
* адаптивная верстка;
* разработка интерактивных элементов на JavaScript;
* подготовка кода для последующей интеграции;
* проектирование поведения интерфейса на промежуточных разрешениях;
* исправление замечаний после тестирования;
* доработка страниц по обратной связи.

---

# ⚙️ Особенности разработки

Во время работы приходится учитывать ряд ограничений проекта.

* Макеты предоставляются только для разрешений **1920px** и **360px**.
* Поведение интерфейса на остальных разрешениях проектирую самостоятельно.
* Код разрабатывается таким образом, чтобы его можно было безопасно интегрировать в существующий коммерческий проект.
* При разработке учитываются возможные конфликты со стилями и компонентами основной кодовой базы.
